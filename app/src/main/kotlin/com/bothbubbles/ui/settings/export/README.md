# Export Settings

## Purpose

Message export UI for exporting conversations to PDF, HTML, or backup format.

## Files

| File | Description |
|------|-------------|
| `ChatSelectionDialog.kt` | Select chats to export |
| `ExportFormatSelection.kt` | Choose export format |
| `ExportOptionsSelection.kt` | Configure export options |
| `ExportProgressDialogs.kt` | Export progress dialogs |
| `ExportScreen.kt` | Main export screen |
| `ExportViewModel.kt` | ViewModel for export |
